# NTT

Carpeta de apoyo para ejercicios y datos del proyecto NTT.

## Contenido

- `coffee_db.parquet`: dataset de ejemplo para analisis exploratorio y modelado.

## Uso sugerido

1. Cargar el dataset con `pandas.read_parquet`.
2. Realizar perfilado inicial (tipos, nulos, outliers).
3. Generar visualizaciones clave en `reports/figures/`.
4. Documentar hallazgos y decisiones en un notebook del repositorio.
